看護スマイル

看護師国家試験対策ノート用のNotion ウィジェット

ウィジェットについて

これらのウィジェットは、Notionページで使用するために特別に設計されています。以下のウィジェットをあなたのNotionページに埋め込んで、学習体験を向上させましょう！

利用可能なウィジェット

プログレスバー

あなたの学習進捗状況を視覚的に表示します。

カスタマイズパラメータ:

  • percentage: 進捗率（例: 50）
  • title: タイトル（例: 学習進捗状況）
  • message: メッセージ（例: 頑張れば、きっと合格できる！）
  • emoji: 絵文字（例: 📚）

フラッシュカード

学習内容を効率的に覚えるためのインタラクティブなフラッシュカード。

カスタマイズパラメータ:

  • term: 用語（例: 心不全）
  • pronunciation: 読み方（例: しんふぜん）
  • category: カテゴリー（例: 循環器疾患）
  • emoji: カテゴリー絵文字（例: 🫀）
  • definition: 定義
  • symptoms: 症状
  • keywords: キーワード
  • memoryTip: 覚え方のコツ

カウントダウンタイマー

試験までの残り日数をカウントダウンします。

カスタマイズパラメータ:

  • title: タイトル（例: 第114回看護師国家試験まで）
  • examDate: 試験日（YYYY-MM-DD形式、例: 2026-02-15）
  • message: メッセージ
  • emoji: 絵文字（例: ⏱️）

モチベーションコーナー

励ましのメッセージや名言、先輩ナースからのアドバイスを表示します。

カスタマイズパラメータ:

  • title: タイトル（例: モチベーションコーナー）
  • emoji: 絵文字（例: 💪）
  • message: 看護ちゃんからのメッセージ
  • quote: 名言テキスト
  • quoteAuthor: 名言の著者
  • advice: 先輩ナースのアドバイス
  • adviceAuthor: 先輩ナースの名前と合格回

学習カレンダー

あなたの学習スケジュールを管理し、効果的な学習計画を立てるのに役立ちます。

カスタマイズパラメータ:

  • month: 表示する月（例: 4）
  • year: 表示する年（例: 2025）
  • scheduleData: 学習スケジュールデータ（JSON形式）
  • tip: 学習のヒント

問題チャレンジ

インタラクティブなクイズで知識をテストし、解説で理解を深めることができます。

カスタマイズパラメータ:

  • category: 問題カテゴリー（例: 循環器疾患）
  • difficulty: 難易度（例: 普通、簡単、難しい）
  • quizData: クイズデータ（JSON形式）
  • timeLimit: 制限時間（秒単位）

東京通勤ナース

可愛らしい看護師キャラクターの一日を通して、楽しく看護知識を学べます。

カスタマイズパラメータ:

  • timeOfDay: 時間帯（例: morning, day, evening, night）
  • message: キャラクターのセリフ
  • factIndex: 表示する看護師雑学のインデックス

科目別サマリー

各科目の重要概念、関連図、押さえるポイント、覚え方のコツなどを一目で把握できます。

カスタマイズパラメータ:

  • subject: 科目（例: circulation, respiratory, digestive）
  • mastery: 習熟度（例: 65）
  • subjectData: 科目データ（JSON形式）

使い方

  1. 使用したいウィジェットの「埋め込みコードをコピー」ボタンをクリックします。
  2. Notionページで、「/embed」と入力し、「Embed」ブロックを選択します。
  3. 「Embed link」フィールドにコピーしたURLを貼り付けます。
  4. パラメータをカスタマイズするには、URLの末尾に「?パラメータ名=値」の形式で追加します。複数のパラメータを設定する場合は「&」で繋ぎます。

例:

https://hybirdss.github.io/Nursing_Smile/widgets/progress-bar/index.html?percentage=75&title=基礎看護学&emoji=📝
